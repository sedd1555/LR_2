1.Git-розподілена система керування версіями написана мовою С, командна оболонка на UNIX, Perl, Tcl, Python та C++.
2.Git має три основних стани, в яких можуть перебувати ваші файли: збережений у коміті (commited), змінений (modified) та індексований (staged)
3.Команда git add додає вміст робочого каталогу в індекс для наступного комміту.
4.Команда git status повертає інформацію про поточний стан репозиторія: що нового, що змінилося тощо.
5.Команда git commit відображає стан нашого репозиторія у момент, коли він був зроблений.
6.Команда git init строрює репозиторій у папці
7.Треба писати зміни які відбувалися у файлі
8.Щоб побачики список комітів теба прописати команду git logo
9.Команда git checkout використовується для переходу між гілками. Вона також може бути використана для переходу між комітами (досить часто у git одна команда може робити багато не спільних між собою функцій).
